# Premiere Presets — NÃO PORTÁVEIS

Mesma situação de `../ae-presets/`. Premiere `.prfpset` / `.mogrt` são formatos
proprietários Adobe. Não rodam no editor AIOX (web + FFmpeg).

Pra trazer um estilo de Premiere pro editor:
1. Recrie o look em CSS (preview) — pasta `../captions/`
2. Recrie em ASS (export FFmpeg) — pasta `../subtitle-styles/`
