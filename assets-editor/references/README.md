## Onde buscar (quando ativar Fase 2 / research web)

- **Google Fonts** — Kanit, Anton, Archivo Black, Bebas Neue, Montserrat, Inter, Oswald, Orbitron, Playfair Display, Rajdhani (todas SIL OFL, uso comercial liberado)
- **LottieFiles** — procurar tags `text-animation`, licenças CC0/CC-BY
- **GitHub** — queries sugeridas: `animated captions css`, `subtitle ass styles`, `kinetic typography css`
- **CodePen** — buscar tag `text-animation` filtrando por licença MIT
- **Animate.style** — MIT, catálogo completo de keyframes CSS

## Assets em outros formatos (não portáveis pro editor web)

- After Effects `.ffx` / `.aep` — proprietário, exige AE desktop
- Premiere `.prfpset` / `.mogrt` — proprietário, exige Premiere desktop
- DaVinci `.drfx` — proprietário, exige Resolve

Esses ficam como **inspiração visual** apenas. Pra trazer pro editor web, o estilo
precisa ser recriado em CSS (preview) + ASS (export). Ver `../captions/` e
`../subtitle-styles/` pra exemplos já convertidos.
