# AE Presets — NÃO PORTÁVEIS

Esta pasta está **intencionalmente vazia**.

Arquivos After Effects (`.ffx`, `.aep`, `.jsx`, `.mogrt`) são formatos proprietários
da Adobe que só funcionam dentro do After Effects desktop. O editor AIOX roda via
**HTML/CSS no browser** e exporta via **FFmpeg com ASS subtitles** — nenhum desses
aceita binários AE.

## Se você quer usar estilo AE como inspiração

1. Abra o preset no AE
2. Screenshot dos keyframes
3. Recrie o equivalente em CSS + ASS nas pastas `../captions/` e `../subtitle-styles/`

## Fontes públicas de AE presets grátis (só pra inspiração visual)

- **VelosoFX (YouTube)** — tutoriais com presets grátis
- **EasyEdit (github.com/Nixq/EasyEdit)** — MIT
- **RubberHose** — pago

Não baixe binários AE aqui — eles não rodam no editor.
