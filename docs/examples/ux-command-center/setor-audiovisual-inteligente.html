Dois Motores Independentes
Operam separados. Podem se alimentar mas nao dependem um do outro.
Cortes Inteligentes
Cerebro Criativo
Recebe video bruto, analisa, transcreve, segmenta em blocos, identifica os melhores cortes por categoria, estima engagement e aguarda aprovacao humana. Aprende com cada decisao.
Ingestao→ FFprobe→ Transcricao→ Segmentacao→ Cortes→ Aprovacao
# Pipeline completo
node bin/av-cortes.js video.mp4

# Aprovar cortes
node bin/av-cortes.js <id> approve-all

# Aprender
node bin/av-cortes.js <id> learn
Escala de Criativos
Motor de Producao
Recebe cortes de qualquer origem, monta o video, aplica legendas inteligentes, branding, valida qualidade, renderiza em batch e organiza outputs prontos pra publicar.
Montagem→ Legendas→ Branding→ Validacao→ Render→ Output
# Produzir aprovados
node bin/av-escala.js <id>

# Gerar variacoes
node bin/av-escala.js <id> variacoes

# Ver outputs
node bin/av-escala.js <id> output

Features Complementares
Brand Catalog, Sugestoes de Melhoria, Performance Analytics, Batch Processing
Brand Catalog
Catalogo de marcas com presets de identidade visual. Cada marca com logo, cores, overlay, estilo de legenda, +18.
node bin/av-brand.js add "Reals" --logo /logo.png
Sugestor de Melhorias
Analisa cortes e sugere ajustes de hook, ritmo, duracao, estrutura. Compara com padroes aprendidos.
node bin/av-suggest.js <project-id>
Performance Analytics
Registra metricas pos-publicacao (views, retencao, shares). Ranking de cortes. Correlacao formato vs resultado.
node bin/av-performance.js <id> analyze
Batch Processing
Ingestao em lote. Processa pasta inteira de videos. Cada video vira projeto independente.
node bin/av-batch.js /pasta-com-videos/
