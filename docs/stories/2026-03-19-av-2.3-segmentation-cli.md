## Contexto

Criar engine de segmentacao: analisar transcricao e metadados para dividir video em blocos logicos (intro, hook, content, story, cta, transition, outro). Usa heuristicas de pausa, mudanca de topico e energia.

## Acceptance Criteria

### Segmentation
- [x] Analisa transcricao e detecta pausas longas (>2s)
- [x] Identifica mudancas de topico por keywords
- [x] Classifica blocos por tipo (intro/hook/content/story/cta/transition/outro)
- [x] Calcula energy level por bloco (high/medium/low)

### Output
- [x] Salva analysis/segments.json
- [x] Cada bloco tem: id, type, start, end, title, excerpt, energy_level
- [x] Total de blocos e duracao total

### CLI
- [x] Comando `node bin/av-segment.js <project-id>` funciona
- [x] Mostra resumo dos blocos no stdout
