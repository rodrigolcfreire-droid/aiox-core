**Phase:** 1 of 7 (Foundation)
**Status:** Done

## Contexto

Criar o schema `audiovisual` no Supabase com 7 tabelas para suportar o pipeline de producao: projetos, videos, cortes, aprovacoes, outputs, presets e aprendizados. Seguir padrao das migrations existentes.

- [x] Adicionado ao array de schemas no Supabase config

### Tabelas
- [x] `audiovisual.project` — Projetos de producao
- [x] `audiovisual.media_asset` — Videos originais e processados

### Campos Obrigatorios por Tabela

#### audiovisual.project
- id UUID PK
- name TEXT NOT NULL
- source_url TEXT
- source_type TEXT (upload/drive/url)
- status TEXT (created/analyzing/analyzed/producing/done)
- metadata_json JSONB
- created_at TIMESTAMPTZ
- updated_at TIMESTAMPTZ

#### audiovisual.media_asset
- id UUID PK
- project_id UUID FK → project
- type TEXT (source/assembled/subtitled/branded/final)
- file_path TEXT
- duration_seconds NUMERIC
- resolution TEXT
- codec TEXT
- file_size_bytes BIGINT
- metadata_json JSONB
- created_at TIMESTAMPTZ

#### audiovisual.cut
- id UUID PK
- project_id UUID FK → project
- category TEXT (viral/autoridade/educativo/storytelling/cta/bastidores/tendencia)
- objective TEXT
- start_time NUMERIC
- end_time NUMERIC
- duration_seconds NUMERIC
- blocks JSONB (array de block IDs)
- engagement_score NUMERIC
- format TEXT (9:16/16:9/1:1)
- platform JSONB (array de plataformas)
- status TEXT (suggested/approved/rejected/edited)
- created_at TIMESTAMPTZ

#### audiovisual.approval
- id UUID PK
- cut_id UUID FK → cut
- project_id UUID FK → project
- decision TEXT (approved/rejected/edited)
- feedback TEXT
- decided_by TEXT
- decided_at TIMESTAMPTZ

#### audiovisual.output
- id UUID PK
- project_id UUID FK → project
- cut_id UUID FK → cut
- media_asset_id UUID FK → media_asset
- title TEXT
- description TEXT
- tags JSONB
- drive_url TEXT
- download_url TEXT
- status TEXT (rendering/rendered/published)
- quality_score NUMERIC
- created_at TIMESTAMPTZ

#### audiovisual.preset
- id UUID PK
- name TEXT NOT NULL
- type TEXT (branding/legenda/format)
- config JSONB
- is_default BOOLEAN
- created_at TIMESTAMPTZ

#### audiovisual.learning
- id UUID PK
- project_id UUID FK → project
- pattern_type TEXT
- pattern_data JSONB
- confidence NUMERIC
- created_at TIMESTAMPTZ
