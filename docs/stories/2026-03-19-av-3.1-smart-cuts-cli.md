# Story AV-3.1: Smart Cuts — Detection + Suggestion

**Epic:** Central Audiovisual (EPIC-AV)
**Phase:** 3 of 7 (Intelligence)
**Status:** Done
**Date:** 2026-03-19
**Depends On:** AV-2.3

## Contexto

Criar engine de cortes inteligentes: analisar blocos segmentados, combinar em cortes potenciais, classificar por categoria (viral/autoridade/educativo/etc), estimar engagement score, e sugerir cortes otimizados para diferentes plataformas.

## Acceptance Criteria

### Cut Detection
- [x] Analisa segments.json e combina blocos em cortes
- [x] Gera cortes com duracao otimizada por plataforma
- [x] Classifica por categoria (viral/autoridade/educativo/storytelling/cta/bastidores/tendencia)
- [x] Calcula engagement score (0-10)

### Platform Optimization
- [x] Reels/TikTok: 15-60s, formato 9:16
- [x] YouTube Shorts: 15-60s, formato 9:16
- [x] Feed Instagram: 30-90s, formato 1:1 ou 4:5
- [x] YouTube: 60-180s, formato 16:9

### Output
- [x] Salva cuts/suggested-cuts.json
- [x] Cada corte tem: id, category, objective, blocks, timestamps, engagement_score, format, platform

### CLI
- [x] Comando `node bin/av-cuts.js <project-id>` funciona
- [x] Mostra cortes sugeridos no stdout
